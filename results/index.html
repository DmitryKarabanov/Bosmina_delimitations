📊 Analysis Results

This directory contains all outputs from the integrative delimitation analysis of Bosmina (Cladocera: Bosminidae).

Contents

Delimitation Results

Summary tables, congruence matrices, and method-specific outputs from all species delimitation methods (bGMYC, GMYC, ABGD, ASAP, Ward, KoT, PTP, CAOS).

For example, bGMYC_delimitation_results.csv — Results of bGMYC4 analysis with MOTU assignments and posterior probabilities.

Statistics

Genetic diversity metrics including haplotype diversity, nucleotide diversity, Watterson's theta, and Tajima's D for all delimited clusters.

For example, bosmina_full_popgen_stats.csv — Comprehensive genetic diversity statistics for all populations and clusters.

Phylogenetic Trees

Newick, Nexus and high-resolution illustrations of tree files with clade annotations. Includes maximum likelihood trees from IQ-TREE and Bayesian trees from BEAST2.

For example, Bosmina_ML.tree — Maximum likelihood phylogenetic tree built with IQ-TREE.
