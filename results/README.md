#  Analysis Results

This directory contains all outputs from the integrative delimitation analysis.

## Subdirectories

**[Delimitation Results](delimitation/)**

Summary tables, congruence matrices, and method-specific outputs.

**[Statistics](stat/)**

Genetic diversity.

**[Phylogenetic Trees](trees/)**

Newick, nexus and high-resolution illustrations of tree files with clade annotations.

## Featured Files

- **`delimitation/bGMYC_delimitation_results.csv`** — Results of bGMYC4 
- **`stat/bosmina_full_popgen_stats.csv`** — Genetic diversity
- **`trees/Bosmina_ML.tree`** — IQ-TREE phylogenetic tree
