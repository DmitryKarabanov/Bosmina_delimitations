✨ Interactive Visualizations

This directory contains all interactive HTML visualizations supporting the integrative species delimitation analysis of Bosmina. Each visualization is a self-contained HTML file that can be opened directly in any modern web browser. No special software or internet connection is required.

Open locally

If you prefer to work offline or need faster loading, download the visualization files to your computer:

  1. Download files from GitHub:

    Go to the interactive directory

    Download the .html file you want (e.g., bGMYC_interactive_heatmap.html)

    Download the corresponding resource directory (e.g., bGMYC_interactive_heatmap_files/)

  2. Keep the folder structure:

    Make sure the .html file and its _files/ directory are in the same folder. The HTML file depends on JavaScript and CSS files stored in the directory.

  3. Open in browser:

    Double-click the .html file to open it in your default web browser (Chrome, Firefox, Edge, or Safari).

  4. Interact with the visualization:

    Zoom: Use mouse wheel or pinch gesture

    Pan: Click and drag to move around

    Hover: Move cursor over elements to see tooltips with details

    Export: Use the camera icon in Plotly toolbar to save as PNG

💡 If you want to download all at once, click the green Code button on the main repository page and select "Download ZIP". Extract the archive and navigate to the interactive/ folder.

Featured Visualization

Our main interactive figure combines two synchronized views:

Left Panel: Clade-Colored Phylogeny

  • Branches colored by major clades.
  • Hover any branch to see taxon name / clade assignment / branch length.

Right Panel: Agreement Matrix

  • Heatmap showing pairwise agreement between delimitation method(s).
  • Color scale from 0% agreement to 100% agreement.
  • Hover any cell to see taxon names / agreement percentage.

Both panels share the same Y-axis, so each row in the matrix corresponds exactly to a tip on the tree.

Troubleshooting

Q: The visualization doesn't load.

A: Try a different browser or clear your cache. Some corporate networks block large HTML files.

Q: Hover tooltips don't appear.

A: Make sure JavaScript is enabled in your browser settings.

Q: The file is too large to view on GitHub.

A: Download the file locally — GitHub has size limits for inline HTML rendering.
