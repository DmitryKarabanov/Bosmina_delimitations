## 📚 About This Study

**Organisms:** *Bosmina* spp. (Cladocera: Bosminidae)  
**Methods:** Integrative species delimitation using multiple independent approaches  
**Clades analyzed:** *Eubosmina*, *Liederobosmina*, *Lunobosmina*, Colombian clade, and others

### Methods Used
- ABGD (Automatic Barcode Gap Discovery)
- ASAP (Assemble Species by Automatic Partitioning)
- bPTP (Bayesian Poisson Tree Processes)
- GMYC (Generalized Mixed Yule Coalescent)
- PTP (Poisson Tree Processes)
- And other delimitation approaches
