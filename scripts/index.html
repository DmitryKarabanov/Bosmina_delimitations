Delimitation Methods

ABGD_like_delimitation.R — ABGD-like algorithm with recursive partitioning. Searches for barcode gap in distance distribution and performs hierarchical clustering with two distance variants (patristic & K80).

ASAP-like_delimitation.R — ASAP-like method comparing two distance metrics. Hierarchical clustering with automatic threshold optimization maximizing intra/ to inter-cluster distance ratio (patristic & pairwise K80 variants).

bGMYC4_interactive.R — Bayesian General Mixed Yule-Coalescent model with interactive visualization. Uses our bGMYC4 package.

CAOS_delimitation.R — Universal CAOS (Character Allowing Separation Of groups) pipeline. Compares 5 delimitation schemes (Ward, ASAP, bGMYC, KoT, bPTP), searches for diagnostic characters, and identifies conflicts between methods.

Delimitation_heatmap_interactive.R — Interactive comparison charts of all delimitation methods + phylogenetic tree. Branch coloring by bGMYC MOTUs, matrix showing results from all methods side-by-side.

GMYC-delimitation.R — GMYC (General Mixed Yule-Coalescent) with single and multi-threshold models. Includes ultrametrization via ape::chronos() and outgroup filtering.

Heatmap_CoMa.R — Interactive agreement matrix (CoMa) of all methods + colored tree. N×N matrix quantifying consistency between all delimitation methods, combined with phylogenetic tree visualization using ggtree and plotly.

KoT-delimitation.R — KoT (K over Theta ratio) post-hoc validation of locMin scheme. Calculates inter-cluster genetic distances (K80) and intra-cluster diversity (Watterson's θ), applies three thresholds (t=4, 6, 8) for standard/strict/very strict criteria.

locMin_delimitation.R — Calculation of local minima (locMin) in distance distribution. Two variants: patristic distances from tree and pairwise K80 distances from sequences.

PTP_parser_webRes.R — bPTP results parser. Reads output from Bayesian PTP web server and converts it into structured CSV format with MOTU assignments.

RefinedThreshold_delimitation.R — Refined Fixed Threshold method. Single-linkage clustering with validation and recursive splitting for problematic clusters. Two variants: patristic (threshold=0.10) and K80 (threshold=0.03).

Ward_delimitation.R — Ward's hierarchical clustering method with automatic threshold selection via local minima search. Two distance variants: patristic and K80.

Requirements

  • R >= 4.5

    Packages: ape, dplyr, tidyr, etc. and Bioconductor packages for phylogenetic analysis

  • Python >= 3.1

    Packages: numpy, scipy, pyvis, etc.

Usage

For bGMYC delimitation you need our bGMYC4 package and R≥4.5: bGMYC4_interactive.R

For nice haplotype network use Anaconda / Python≥3.1: network_new.py

Output

Scripts generate results in the Results directory.
